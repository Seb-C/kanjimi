Read any Japanese website

Kanjimi is a web-application and a browser extension that works with any Japanese website. It automatically adds useful information to the text.

Improve your vocabulary

In order to learn, a complete translation of the text is not helpful. That is why we focus on providing the meaning and pronunciation for every word.

Learn more
 

Study the words and Kanjis

Integrated dictionary

Click on a word to show a detailed description about it. The meanings are displayed separately for each possible reading of a word.

Supports multiple languages

The translations can be displayed in multiple languages, including 🇬🇧 English, 🇫🇷 French, 🇪🇸 Spanish, 🇩🇪 German and many others.

Information about the Kanjis

We also display information about the Kanjis, including the readings and meanings.
